Applications de gestion pour RedStamp SRL Italia

Projet Stage International de 1ere année
Technologie Python, SQL
Année 2020
Mon rôle Développeur

Au cours de mon stage de première année en usine de manufacture de pièces automobiles en Italie, j'ai développé deux applications en python. La première afin d'automatiser le traitement des factures et la gestion d'une base de données des matériaux et pièces qui entrent et sortent de l'usine.

Le second programme permet de prendre un fichier de modèle 3D, de demander pour chaque trou certains paramètres à l'utilisateur et de retourner un fichier compréhensible pour la machine qui va ensuite construire la pièce modélisée.

Les deux applications ont été développées en Python en quelques semaines. De nombreux échanges avec le client ont été nécessaires pour définir exactement ses besoins, se fut donc ma première expérience de développement pour un client n'ayant pas de compétences dans ce domaine.

Automatisation du traitement des factures

Rentrer une facture

Le client peut ajouter une facture. Le programme analyse automatiquement le PDF, extrait les informations et les enregistre dans l'excel de gestion, en distinguant les éléments achetés de ceux vendus.

Gestion des clients

Ajouter ou supprimer manuellement un client avec son numéro de TVA.

Gestion des stocks

Ajouter ou supprimer manuellement une certaine quantité d'un article dans l'inventaire.




Outil de conversion de fichiers 3D en code machine

Rentrer un fichier 3D

Le client rentre un fichier, puis choisis le type de machine, Fanuc ou She Hong

Choix du type de trou

Pour chaque trou dans le modèle 3D, le client sélectionne s'il s'agit d'un perçage, d'un filetage ou d'un alésage.

Choix des paramètres

Selon le type de trou choisi, le client saisit ensuite les paramètres requis.
