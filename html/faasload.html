Visualisation et analyse de performance d’applications FaaS

Projet Stage de 2eme année à l'Institut Polytechnique de Paris
Technologie Python, Plotly, PosgreSQL, SQLAlchemy, Docker, OpenWhisk
Année 2022
Mon rôle Développeur

Au cours de mon stage de deuxième année au sein de Télécom SudParis, j'ai travaillé sur l'application FaaSload. FaaSLoad est un logiciel d’injection de charge et un moniteur de performance conçu pour le FaaS : Function-as-a-Service, un service de cloud computing qui fournit une plate-forme permettant aux clients de développer, d'exécuter et de gérer des fonctionnalités d'application sans la complexité de la construction et de la maintenance de l'infrastructure généralement associée au développement et au lancement d'une application.

J'ai pu améliorer la base de donnée existante ainsi que concevoir une application de Visualisation des données. Cette application se devait d'être très modulable afin de pouvoir afficher plusieurs graphs en même temps et de les comparer. De même, le code se voulait lui aussi très modulaire afin que notre tuteur de stage puisse facilement rajouter lui-même des graphs et des variables au projet si nécéssaire.

Changement complet de backend

J'ai collaboré avec un autre élève pour refactorer totalement le backend du stockage des données. De mon côté, j'ai intégré SQLAlchemy pour améliorer la communication entre la base de donnée et le code python. En effet, SQLAlchemy facilite la liaison entre Python et les bases de données SQL en convertissant automatiquement les appels de classes de Python en instructions SQL. Mon collaborateur s'est chargé de remplacer MariaDB par PostegreSQL.

Des outils de filtrage

Nous avons développé des méthodes permettant de filtrer différents types de données tout en offrant la possibilité de cumuler plusieurs filtres simultanément.

Un nouvel outil de visualisation

Nous avons utilisé Dash de Plotly, un framework Python permettant de créer des applications web interactives pour la visualisation de données. Grâce à cet outil, nous avons pu facilement afficher et manipuler des courbes et des graphiques pour mieux analyser les données
