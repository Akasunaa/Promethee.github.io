Bubbles, Please

Game Jam Global Game Jam 2025
Theme Bubbles
Technologie Unity
Mon rôle Graphiste, Développeur

Incarnez un mignon petit vendeur de Bubble Tea dans la boutique de Bubble Tea la plus mignonne de l'univers ! Mettez vos ingrédients dans toutes vos machines pour créer les boissons les plus agréables pour vos merveilleux clients. Appréciez le style de vie délicieux d'une boutique de Bubble Tea ! ... Mais est-ce vraiment le cas ?
